## 🧠 Behind the Technology

This tool uses a multi-stage approach to handle your information. 

First, it breaks your long document into smaller parts. This makes it easier for the system to index the content. It stores these parts in a vector database. A vector database is a digital library that organizes information by meaning rather than just keywords. 

When you ask a question, the system searches the database for matches. It looks for sections that share the same context as your query. This step happens in several stages to ensure the results stay relevant. An advanced language model then reads these parts to synthesize a helpful response. This process mimics how a human researcher gathers facts and builds a summary.

## 🔧 Managing Your Library

You can remove documents you no longer need. Open the sidebar and click the Manage tab. This view lists all files you have processed. Click the trash icon next to a file name to delete it from the vector database. This frees up space on your hard drive. 

If the system runs slow, you can limit the number of active documents. Keep only the files relevant to your current project in the active list.

## 💡 Troubleshooting

If you encounter issues, try these steps first.

*   **System does not open:** Ensure you have administrator rights on your computer.
*   **Search results are empty:** Check if your document contains valid text. If you uploaded a scanned PDF image, the system might not read it. Use a document that contains selectable text.
*   **Slow performance:** Close other heavy programs on your computer to free up memory.
*   **Connection errors:** Ensure your firewall allows local connections. The software only communicates with your web browser on your own machine.
*   **Update issues:** Download the newest version from the link above and run the installer again. It overwrites the old version and keeps your settings intact.
